Windows Git Yubikey

Posted by Manses notes on Friday, January 14, 2022

I prefer to use git via ssh. And I like to store my ssh private key on a yubikey. Using linux, this setup is trivial, just follow the direction from the linked manuals from yubico

But what about my windows client?

Well, follow the guide for windows yubico’s windows guide and dont forget use the putty plink option during the git installation. I dit forget to set it and it took embarassingly long to find that error.
